# XP-Games
I tried to make games that work on Windows XP, weak Pentium, IE6, IE8.
If someone can check on their PC.
You'll also need to run the launcher, and most games may be in Russian, as this was a project for a school in Russia. The launcher will detect your PC and give you the most suitable versions of the games based on quality (particles, pixels) and code age (from XP to modern versions).
